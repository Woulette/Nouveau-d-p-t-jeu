# Standard visuel non négociable

La référence est un MMORPG fantasy 2D chaleureux, dense et lisible sur téléphone : grands arbres ombrés, plusieurs variantes d’herbe, fleurs, pierres, chemins, clôtures, bâtiments détaillés, personnage animé et interface sombre bordée d’or.

Avant toute fusion sur `main` :

1. aucune animation existante ne disparaît ;
2. aucun asset final n’est remplacé par un emoji ou un rectangle ;
3. la carte ne présente pas un grand aplat vert vide ;
4. le personnage reste lisible sur les quatre formats mobiles testés ;
5. les effets de bâton, fronde et magie restent visibles ;
6. une capture comparative est produite ;
7. les tests fonctionnels et visuels sont passés.
