# Nettoyage des anciens prototypes

La base officielle ne dépend d’aucun des anciens projets ou pages de test.

Sont exclus de cette fondation :

- pages « Solenne Alpha fonctionne » ;
- anciens projets Vercel temporaires ;
- sprites en emojis ;
- personnages constitués de rectangles simplifiés ;
- carte verte vide ;
- fichiers absolus `/mnt/data/...` ;
- assets récupérés depuis un autre jeu ;
- faux joueurs présentés comme un vrai MMO.

Le dépôt officiel reste `Woulette/Nouveau-d-p-t-jeu`. La cible Vercel officielle reste `chroniques-de-solenne`.
