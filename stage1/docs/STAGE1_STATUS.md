# Statut de l’étape 1 — Fondation

Version : `0.8.0-alpha.1`

## Fondation consolidée

- client statique reproductible dans `stage1/` ;
- interface mobile paysage ;
- déplacement tactile case par case et pathfinding ;
- collisions avec les obstacles ;
- ciblage, approche et combat automatique ;
- bâton, fronde et orbe magique ;
- XP générale, XP de classe et quatre maîtrises ;
- vitesse `100 + (niveau général - 1)` ;
- monstres, butin, inventaire, équipement, mort et réapparition ;
- mentors Épéiste, Archer et Mage ;
- sauvegarde locale ;
- PWA et cache hors ligne ;
- atlas pixel art séparés et versionnés dans `assets/` ;
- contrôle automatique sans emoji ni placeholder.

## Validation

La publication ne doit être promue vers `main` qu’après réussite des contrôles locaux et du test distant Vercel sur les formats 667×375, 844×390, 896×414 et 932×430.

La phase artistique suivante doit améliorer les assets sans supprimer une animation, un système de jeu ou un niveau de détail déjà validé.
