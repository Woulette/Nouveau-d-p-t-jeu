# Chroniques de Solenne

Dépôt officiel du prototype mobile 2D en pixel art.

## Alpha jouable

- déplacement tactile case par case avec recherche de chemin ;
- ciblage d’un monstre, approche automatique et attaque automatique ;
- Aventurier avec bâton, fronde et orbe magique ;
- XP générale, XP de classe et maîtrises de combat ;
- butin, or, inventaire, statistiques, mort et réapparition ;
- atlas pixel art originaux pour le héros, les monstres, les objets et les icônes ;
- interface mobile en mode paysage ;
- sauvegarde locale.

Le véritable serveur MMO autoritaire sera branché après validation de cette tranche verticale.

## Branches

- `main` : version stable publiée sur Vercel ;
- `develop` : développement et tests avant publication.
